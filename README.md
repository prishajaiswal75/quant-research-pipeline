

A quantitative research framework for developing, evaluating, and explaining systematic trading strategies using classical machine learning and reinforcement learning.

## Objectives

- Feature engineering on financial time series
- Supervised learning baselines
- Reinforcement learning for trading
- Risk-adjusted backtesting
- Explainable AI using SHAP
- Research experiment tracking
